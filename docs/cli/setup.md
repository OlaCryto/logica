# `logica setup`

Initialize `~/.logica/logica.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
logica setup
logica setup --workspace ~/.logica/workspace
```

To run the wizard via setup:

```bash
logica setup --wizard
```
